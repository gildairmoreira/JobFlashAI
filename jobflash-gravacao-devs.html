// 01 — Antes de gravar
Setup Mínimo Viável

Você não precisa de câmera, iluminação profissional ou editor de vídeo pago. Precisa só disso:

🖥️

Gravação de Tela

Use OBS Studio (grátis) ou Loom (grátis com limite). Configure resolução 1080×1920 (vertical) ou grave em 1920×1080 e vire no editor.

🎤

Áudio

Microfone do fone de ouvido já serve. Grave num cômodo silencioso. Teste antes: fale em voz normal e escute. Sem eco = bom.

✂️

Edição

Use CapCut (grátis, celular ou PC). Perfeito para Shorts/TikTok. Já tem legendas automáticas em pt-BR. Essencial: ative "Auto Captions".

🎨

Texto na Tela

No CapCut, adicione texto grande para quem assiste sem som. Fonte Bold, contraste alto. Regra: cada fala importante = 1 texto na tela.

Dica de dev: Prepare o JobFlashAI aberto em modo demo no browser antes de gravar. Deixe um currículo de TI genérico já carregado e uma vaga de desenvolvedor real do LinkedIn copiada. A demo precisa parecer natural, não roteirizada.

// 05 — Tutorial
Como gravar a tela no OBS

Do zero ao arquivo de vídeo em ~20 minutos

Baixar o OBS Studio

obsproject.com → Download → Windows/Mac/Linux. Instale com configurações padrão.

Configurar saída de vídeo

Configurações → Vídeo → Resolução base: 1920×1080. FPS: 30. Output: MP4. Qualidade: CRF 18.

Adicionar fontes de captura

Na cena principal: clique "+" em Fontes → "Captura de Tela" (para capturar o browser com o JobFlashAI). Adicione também "Captura de Áudio" para o microfone.

Fazer um teste antes

Grave 30 segundos, pare, assista. Verifique: áudio claro? Tela nítida? Cursor visível? Ajuste o que precisar.

Gravar com o JobFlashAI aberto

Abra o browser com jobflash.netlify.app em tela cheia. Tenha o currículo de dev + descrição de vaga já prontos. Clique "Iniciar Gravação" → execute o roteiro → "Parar Gravação".

Importar no CapCut e editar

Abra o CapCut → "Novo Projeto" → importe o .mp4. Corte silêncios, adicione textos, ative "Auto Legenda" em pt-BR. Exporte em 1080p.

Virar para formato vertical (9:16)

No CapCut: proporção 9:16. O screen record vai ficar centralizado. Adicione fundo preto ou desfocado acima/abaixo. Pronto para Shorts/TikTok/Reels.
